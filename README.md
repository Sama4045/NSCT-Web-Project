# NSCT Student Registration System

A simple Student Registration System developed using **HTML, CSS, PHP, and MySQL**. This project was created as part of my Web Development practice for the NSCT course.

## Features

- Student registration form
- Stores student information in a MySQL database
- Database connectivity using PHP
- Simple and clean user interface
- Data is saved successfully after form submission

## Technologies Used

- HTML5
- CSS3
- PHP
- MySQL
- XAMPP
- VS Code

## Project Structure

```
NSCT/
│── index.php
│── insert.php
│── db.php
│── style.css
```

## Database

Database Name:

```
nsct
```

Table Name:

```
students
```

The following information is stored:

- Name
- Father Name
- CNIC
- Date of Birth
- Address
- CNIC Issue Date
- CNIC Expiry Date

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into `htdocs`.
4. Create a database named `nsct`.
5. Create the `students` table.
6. Open:

```
http://localhost/nsct/
```

7. Fill in the form and submit the data.
